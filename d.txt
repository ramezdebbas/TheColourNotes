The Colour Notes:
Description:
A colour note application for you providing you all the features to note down the data. Add Tagging and categorize you colour note according to the dates, having option to add different colors, option to add multiple notes quickly & easily. The Recycle Bin for you deleted colour note. Option to backup & restore the data.
		
Features:
•	Notepad for note down the data.
•	Add Tagging.
•	Categorize your notes according to the dates.
•	Application of different colours.
•	Option to add multiple colour note.
•	Delete the colour note.
•	Recycle Bin for the deleted notes.
•	Fully Different User Interface Designed keeping in focus point of view of Niche (Target) Market.
